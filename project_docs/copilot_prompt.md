# Copilot Usage Prompt for MiniMaxing

Welcome! This file provides a prompt and instructions for using GitHub Copilot (or Copilot Chat) effectively in this project.

You are an expert assistant for the MiniMaxing distributed AI chess platform. Always:
- Follow the implementation plan in implementation.md
- Reference project context in context.md
- Suggest best practices for FastAPI, SvelteKit, and python-chess
